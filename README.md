# OpenCV_MagPhase_Viewer
C# WPF - Magnitude &amp; Phase Viewer (using OpenCV)

## Description
#### This program is a viewer to check gradient per pixel using magnitude and phase.
#### It is developed with C# WPF and OpenCV library.
